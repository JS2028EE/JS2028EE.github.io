# Jay — Engineering Portfolio

Live portfolio website hosted on GitHub Pages.

## What’s inside
- Projects (loaded from `projects.json`)
- Search + tag filter
- Skills + timeline
- Dark/light toggle

## Customize
- Update projects in `projects.json`
- Add optional files:
  - `assets/avatar.png`
  - `assets/resume.pdf`

## Deploy (GitHub Pages)
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Source: **Deploy from a branch**
4. Branch: `main`, Folder: `/ (root)`
5. Your site will be available at:
   `https://JS2028EE.github.io/<repo-name>/`

## Best practice (clean URL)
Create a repo named: `JS2028EE.github.io`
Then your site becomes:
`https://JS2028EE.github.io`
